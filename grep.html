Ejemplo de uso del comando grep

Buscar una palabra en un archivo:

grep "texto" archivo.txt

Buscar sin distinguir mayúsculas/minúsculas:

grep -i "texto" archivo.txt

Buscar en varios archivos:

grep "texto" *.txt

Mostrar número de línea:

grep -n "texto" archivo.txt

EJEMPLOS DE ERRORES COMUNES

Error: no usar comillas en textos con espacios

grep texto largo archivo.txt

Correcto:

grep "texto largo" archivo.txt

Error: olvidar -i para ignorar mayúsculas

grep "hola" archivo.txt

Correcto:

grep -i "hola" archivo.txt

Error: usar mal comodines

grep "texto" .txt

Correcto:

grep "texto" *.txt
